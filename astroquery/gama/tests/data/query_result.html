SQL Query

The form below allows you to directly submit a MySQL query to the GAMA DR2 database. Consult the schema browser to find out which tables are available in the database. Some example queries are provided for those unfamiliar with MySQL. For complex queries you may want to consider using the Query Builder.

The only limit placed on queries launched with the form below is that they must execute within 10 minutes. If your query times out then you should download the relevant tables from one of the file servers and perform your query locally (e.g. using TOPCAT).

 

Format of results file: CSV ASCII FITS
